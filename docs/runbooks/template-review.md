# Template review

Use this runbook before using this repository as a base for a real app.

## Areas to review

- product journeys
- local setup
- tests
- automated checks
- environments
- infrastructure
- observability
- deployment notes
- rollback notes
- agent loop reports

## Evidence

- Web app can run.
- API app can run.
- Worker can run or execute a sample job.
- Critical journey has a browser test.
- Local setup works from a clean clone.
- Automated checks run real commands.
- Staging validation exists.
- Health checks and logs exist.

## Decision

Choose one:

- `Ready`
- `Ready with risks`
- `Not ready`
